The HTML <head> Element

Learn more about the <head> element by adding a meta description and favicon.

What Is the <head> Element?

The <head> element contains information about a webpage rather than the visible content that visitors can see. Most of what you place in the head is not shown as regular page content, but it does help the browser understand the page and correctly load important resources.

Think of the head as the page’s behind-the-scenes setup area. It can include the page title, information about character encoding, settings for responsive design, links to CSS files, icons for browser tabs, and other helpful metadata.

What Is the <meta> Element?

The <meta> element is one of the most common elements used inside the <head>. It is used to provide metadata, which means information about the webpage rather than content that appears inside the page itself.

The <meta> element is flexible because it can do different jobs depending on which attributes it uses. For example, you've already used it to set character encoding and viewport behavior, and in this exercise you'll use it again to add a page description.

Things You've Already Been Using in the <head>

Even if you haven't thought about them lately, you've already been using several important elements inside the <head>.

  • <meta charset="UTF-8">
    This tells the browser what character encoding to use so text, punctuation, and special characters display correctly.
  • <title>
    This sets the title of the webpage. It appears in the browser tab and helps identify the page.
  • <meta name="viewport" content="width=device-width, initial-scale=1.0">
    This helps the page display correctly on phones and other mobile devices by controlling the viewport width and zoom behavior.
  • <link rel="stylesheet" href="styles.css">
    This connects the HTML page to an external CSS stylesheet so you can control the visual appearance of the page(s).
  • <style>
    This allows CSS to be written directly inside a single HTML file instead of in a separate stylesheet file.

There are also many other things that can be placed inside the <head>, but some of them are more advanced, more specialized, or outside the scope of this course. For Web Design 1, we're focusing on the head elements that are most practical for beginner websites.

New Things for This Exercise

Meta Description

A meta description is a short summary of what a webpage is about. It is placed inside the <head> and helps describe the page behind the scenes. Search engines like Google may use this text as part of the page’s snippet in search results, so a good description can help people understand what the page is about before they click. This description isn't shown as regular content on the webpage itself, so visitors won't see it when viewing the page.

Favicon

A favicon is the small icon that appears in a browser tab next to the page title. It helps identify and brand a website visually.

For this exercise, you'll link a favicon image file so the browser can display it in the tab. The image file has already been included in the project folder for you.
